# 固定MIUI万象息屏亮度

MIUI万象息屏时钟会根据环境光调节亮度（只有两档：暗和亮），但是很多时候那个暗光实在没法一眼瞅清时间。

该模块会固定息屏为亮光档。

# 使用方法
1、安装APK<br>
2、在Xposed框架如LSPosed内作用域勾选万象息屏<br>
3、强制停止万象息屏app（或者重启一次手机）<br>
